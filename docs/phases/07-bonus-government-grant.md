## 프롬프트

당신은 한국 정부지원사업 사업계획서 작성 컨설턴트입니다. 합격률이 높은 사업계획서를 작성하는 전문가로, 중기부/창업진흥원/TIPS/KOITA 등 주요 사업의 심사 기준을 깊이 이해하고 있습니다.

## 신청자 배경
- 이름: W
- 경력: 한국 벤처캐피털 디렉터 10년+ (보광인베스트먼트)
  - AUM 224.3억원, 50+ 스타트업 투자 (AI/바이오/반도체 등 딥테크)
  - 14개 펀드 운용 경험
- 학력: 서울대학교
  - 임상약리학 박사과정
  - 생물통계학 석사
- 기술 역량: R, SAS, Python (논문 출판 경험)
- 자본: 자기자본 500~1,000만원 (초기 운영비)

## 사업 개요
- **사업명:** DealSync (딜싱크)
- **한 줄 소개:** 섹터별 전문 AI 심사역 6명을 고용하는 VC용 투자심사보고서 자동화 SaaS
- **MVP 상태:** Claude Code로 개발 완료, 베타 테스트 진행 예정
- **차별화 4축:**
  1. 섹터별 전문 AI 에이전트 6명 (BIO/IT/AI/제조/콘텐츠/핀테크)
  2. 회사별 보고서 양식 1:1 픽셀 재현
  3. 풀사이클 (딜소싱→심사→사후관리→LP리포팅)
  4. 셀프서브 가격 공개
- **시장:** 한국 VC 600개 + 액셀러레이터 350개 + CVC 100개 = TAM 1,050개 기관
- **수익 모델:** 월 구독 9.9만~199만원, 플랜별 차등

## 경쟁사 분석 (사업계획서에 명시할 내용)
1. **VCNote (vcnote.com)** - 표준 1페이지 양식, 섹터 차별화 없음
2. **MYSC 메리, 더벤처스 비키, 카카오벤처스 KV파트너스** - 모두 인하우스 전용, 외부 판매 안 함
3. **Skywork** - 글로벌 범용 도구, 한국 VC 도메인 없음

다음 중 가장 적합한 정부지원사업을 추천하고 각각에 맞춤형 사업계획서를 작성해주세요:
- 중기부 초기창업패키지 (최대 1억원)
- 중기부 창업도약패키지 (최대 3억원)

## 작성 요청

### 1단계: 사업 추천 (먼저 답해주세요)
위 정부지원사업 중 W님의 배경과 DealSync 사업에 가장 적합한 사업 TOP 3을 추천하고, 각각의 합격 가능성과 이유를 분석해주세요.

### 2단계: 추천된 TOP 1 사업의 사업계획서 작성
다음 9개 섹션을 표준 양식에 맞춰 작성:

**1. 사업 개요 (요약문)**
- 한 줄 가치 제안
- 해결하려는 문제
- 솔루션 차별성
- 시장 규모와 진입 전략
- 매출 목표 (3년)

**2. 문제 정의**
- 현재 VC 심사 프로세스의 비효율 (정량 데이터 포함)
  - 한 건 심사에 평균 X시간
  - 정보 수집 비중 Y%
  - 양식 맞추기에 Z시간
- 인하우스 AI 개발의 한계 (600개 VC 중 5개사만 자체 AI 보유)
- 경쟁 솔루션의 한계

**3. 솔루션 차별성**
- 4축 차별화 상세 설명
- 경쟁사 비교 매트릭스 (VCNote vs DealSync)
- 진입장벽 (W님의 임상약리학 박사 = BIO 모듈 해자)

**4. 시장 분석**
- TAM/SAM/SOM 계산:
  - TAM: 한국 1,050개 기관 × 평균 ARPU 100만원/월 = 연 1,260억원
  - SAM: 적극 도입 가능 기관 30% = 315개 × 150만원/월 = 연 567억원
  - SOM: 3년차 5% 점유 = 28억원
- 시장 성장 동력 (AI 심사역 트렌드)
- 한국 VC 시장의 디지털 전환 흐름

**5. 비즈니스 모델**
- 구독 5단계 (9.9만/29만/79만/149만/199만원)
- LTV/CAC: Solo 플랜 기준 LTV 200만원 / CAC 30만원 = 6.7x
- 단위경제:
  - 보고서 1건 생성 비용: API 0.5만원 + 인프라 0.1만원 = 0.6만원
  - 평균 ARPU: 50만원/월 (Sector Pro)
  - 총마진: 90%+

**6. 팀 역량**
- 대표자 (W) 강조:
  - VC 10년 경력 = 도메인 권위
  - 임상약리학 박사 = BIO 모듈 해자
  - R/Python/SAS = 직접 개발 가능
- 1인 시작의 정당성:
  - Claude Code로 풀스택 개발 가능
  - 정부지원으로 6개월 내 팀 2명 채용
- 자문단 구성 계획 (VC 선배, 기술 전문가)

**7. 자금 사용 계획 (1억원 가정)**
- 개발: 4,000만원 (40%)
  - AI API 비용 12개월
  - 인프라 (Vercel, Supabase)
  - 외부 데이터 API 라이선스 (FDA, PubMed 등)
- 마케팅/영업: 3,000만원 (30%)
  - 콘텐츠 마케팅 (블로그, LinkedIn)
  - VC 컨퍼런스 부스
  - 베타 고객 무료 → 유료 전환 인센티브
- 운영: 2,000만원 (20%)
  - 사무실 (공유오피스)
  - 법무/세무
- 인건비: 1,000만원 (10%)
  - 외주 디자이너/마케터

**8. 3년 재무 계획**
- Year 1: 손익분기 (월 MRR 1,000만원 = 누적 매출 1.2억)
- Year 2: 흑자 전환 (월 MRR 3,000만원, 인건비 3명)
- Year 3: 시리즈A (월 MRR 1억원, AUM 환산 10억원)
- KPI:
  - Year 1: 유료 고객 50개사
  - Year 2: 200개사 + 한국 시장 점유율 20%
  - Year 3: 500개사 + 동남아 진출

**9. 사회적 가치 (선택)**
- 한국 스타트업 투자 생태계 효율화
- 중소 VC의 의사결정 품질 향상
- 정부 R&D 자금 배분의 정확성 제고
